# Nessus Intelligence Logo Package

## Recommended everyday files

- Website header on a light page: `01_Master_Vector/nessus-primary-horizontal-on-light.svg`
- Website header on a dark page: `01_Master_Vector/nessus-primary-horizontal-on-dark.svg`
- Reports and proposals: `01_Master_Vector/nessus-wordmark-on-light.svg`
- Social avatar and favicon: files in `04_Favicons_App_Icons/`
- Professional printing: PDF or EPS in `02_Print/`
- Microsoft Office and general documents: transparent PNG files in `03_Web/`

## File types

- SVG: editable vector master for web, Figma, Illustrator, Affinity Designer and most design tools.
- PDF: print-ready vector artwork. Text is outlined, so no font installation is required.
- EPS: legacy print and signage format.
- PNG: transparent raster artwork for presentations, documents and web use.
- JPG: flattened artwork for platforms that do not support transparency.
- WebP: compact web format.
- TIFF: 300 dpi CMYK raster for print workflows that specifically request it.
- ICO and app-icon PNGs: browser favicon and application icons.

## Usage

- Use the `on-light` versions on white, pale grey or other light backgrounds.
- Use the `on-dark` versions on Parliament Navy or another very dark background.
- Keep clear space around the logo equal to at least half the height of the N monogram.
- Do not stretch, rotate, add shadows, recolor individual letters or place the logo over busy imagery.
- At very small sizes, use the N monogram instead of the full wordmark.

## Brand colors

- Parliament Navy: #041632
- Navy Surface: #1B2B48
- Nessus Gold: #C6A15B
- Document White: #F7F9FB
